# UI UX Pro Max

一个现代化的 Nuxt 4 项目，集成了最新的前端技术栈。

## 🚀 技术栈

- **Nuxt 4.4.2** - Vue.js 的强大框架
- **Tailwind CSS 4.2.2** - 实用优先的 CSS 框架
- **TypeScript 5.9.3** - JavaScript 的类型超集
- **ESLint 9.39.4** - 代码质量和风格检查工具
- **Vue 3.5.32** - 渐进式 JavaScript 框架

## 📦 安装

```bash
npm install
```

## 🛠️ 开发

启动开发服务器：

```bash
npm run dev
```

访问 http://localhost:3000

## 🏗️ 构建

构建生产版本：

```bash
npm run build
```

预览生产构建：

```bash
npm run preview
```

## 📝 代码检查

运行 ESLint：

```bash
npx eslint .
```

## 📄 项目结构

```
ui-ux-pro-max/
├── app/
│   └── app.vue          # 主应用组件
├── components/          # Vue 组件
├── pages/              # 页面路由
├── public/             # 静态资源
├── server/             # 服务器端代码
├── nuxt.config.ts      # Nuxt 配置
├── tailwind.config.js  # Tailwind 配置
└── eslint.config.mjs   # ESLint 配置
```

## 🎨 特性

- ⚡️ 快速的热模块替换 (HMR)
- 🎨 Tailwind CSS 实用类
- 📘 TypeScript 类型支持
- ✅ ESLint 代码检查
